# 毕业设计项目规划书：基于 YOLOv11 的水下目标检测算法研究与实现

## 1. 项目概览

- **研究方向**：深度学习、水下机器视觉 
- **核心目标**：针对水下环境导致的图像质量退化问题，基于 **YOLOv11** 框架设计一套兼顾检测精度与实时性的水下目标检测方案 。
- **实验数据**：
  - **主数据集**：URPC (Underwater Robot Picking Contest) —— 用于模型训练与核心评估 。
  - **辅助/测试集**：RUIE (Real-world Underwater Image Enhancement) —— 用于验证模型在不同浊度、色彩偏差下的泛化表现 。

## 2. 核心研究任务

### 2.1 水下图像鲁棒性增强研究

- **预处理对比实验**：对比传统物理模型（如暗通道先验）与深度学习增强（如 RUIE 中的增强逻辑）对 YOLOv11 输入特征提取的影响 。
- **数据扩增方案**：针对水下样本不平衡问题，研究 Mixup、Mosaic 及模拟水下光线散射的特殊增强策略 。

### 2.2 YOLOv11 网络结构优化

- **轻量化注意力模块集成**：在 YOLOv11 的 C3k2 或 Neck 部分引入 **ECA (Efficient Channel Attention)** 或 **CA (Coordinate Attention)**，提升模型对低对比度目标的感知能力 。
- **特征融合网络改进**：优化特征金字塔（FPN/PANet）结构，加强深层语义与浅层细节的融合，解决小目标漏检问题 。
- **边界框损失函数优化**：引入 **WIoU v3** 等具备动态聚焦能力的损失函数，加快收敛速度并提升定位精度 。

### 2.3 部署测试与推理效率分析

- **模型导出与转化**：将改进后的模型转化为 ONNX/TensorRT 格式 。
- **推理性能对比**：测试模型在 FP32、FP16 精度下的计算开销（FLOPs）与实时推理速度（FPS），验证其实际部署的可行性 。

## 3. 技术指标设定

| **指标类别** | **评估参数**           | **目标值**                                |
| ------------ | ---------------------- | ----------------------------------------- |
| **精度指标** | 平均精度均值 (mAP@0.5) | $\ge$ 82% (URPC 验证集)                   |
|              | 小目标召回率 (Recall)  | 较 YOLOv11 基线提升 $\ge$ 5%              |
| **效率指标** | 参数量 (Parameters)    | $\le$ 7M (保持轻量化水平)                 |
|              | 推理速度 (FPS)         | NVIDIA RTX 系列显卡下 $\ge$ 60 FPS        |
| **稳健性**   | 跨域泛化性             | 在 RUIE 不同水质子集上的性能波动 $\le$ 8% |

## 4. 实施待办事项 (To-Do List)

- **环境配置**：搭建 PyTorch 2.x + CUDA 12.x 开发环境，安装 Ultralytics 框架。
- **数据准备**：下载 URPC/RUIE 数据集
- **基线训练**：运行 YOLOv11 原始模型训练，记录 mAP、Loss 曲线及混淆矩阵作为 Benchmark 。
- **模块开发**：在源码中注册新的注意力模块及 Loss 函数，编写 YAML 配置文件。
- **消融实验**：逐一开启/关闭改进模块，记录各项指标变化并截图可视化检测效果 。
- **对比分析**：将最终模型与 YOLOv8、YOLOv10 等主流算法在同一数据集上进行横向对比 。
- **部署转化**：编写导出脚本，测试并记录不同量化精度下的推理耗时。
- **论文撰写**：整理实验图表，撰写题目依据、方法论及结果分析部分 。

- **学术成果**：符合学术规范的本科毕业论文，重点突出改进逻辑与实验数据支撑。
